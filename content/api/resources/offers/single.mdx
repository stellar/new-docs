The single offer endpoint provides information on a specific offer.

<Endpoint>

|     |                   |
| --- | ----------------- |
| GET | /offers/:offer_id |

</Endpoint>

<AttributeTable>

- ARGUMENT
  - REQUIRED
  - DESCRIPTION
- offer_id
  - required
  - A unique identifier for this offer.

</AttributeTable>

<CodeExample title="Example Request">

```curl
curl "https://horizon.stellar.org/offers/165563085"
```

</CodeExample>

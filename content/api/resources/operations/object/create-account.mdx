Creates and funds a new account with the specified starting balance.

See the [`Create Account` errors](../../../errors/result-codes/operation-specific/create-account.mdx).

<AttributeTable>

- ATTRIBUTE
  - DATA TYPE
  - DESCRIPTION
- starting_balance
  - string
  - The amount of XLM to send the newly created account.
- funder
  - string
  - The account that funds the new account.
- account
  - string
  - A new account that is funded.

</AttributeTable>

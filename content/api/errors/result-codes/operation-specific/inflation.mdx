These are result codes that communicate success (200) or failure (400) specific to the `Inflation` operation.

Learn more about the [`Inflation` operation](../../../../docs/glossary/operations/inflation.mdx).

<AttributeTable>

- OpSuccess
  - INFLATION_SUCCESS
  - Inflation operation has run successfully
- op_not_time
  - INFLATION_NOT_TIME
  - It's not yet time for the inflation operation to run

</AttributeTable>

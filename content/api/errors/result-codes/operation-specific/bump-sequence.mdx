These are result codes that communicate success (200) or failure (400) specific to the `Allow Trust` operation.

Learn more about the [`Allow Trust` operation](../../../../docs/glossary/operations/allow-trust.mdx).

<AttributeTable>

- OpSuccess
  - BUMP_SEQUENCE_SUCCESS
  - Sequence number has been bumped
- op_bad_seq
  - BUMP_SEQUENCE_BAD_SEQ
  - `bumpTo` is not within bounds

</AttributeTable>

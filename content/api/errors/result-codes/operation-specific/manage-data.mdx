These are result codes that communicate success (200) or failure (400) specific to the `Manage Data` operation.

Learn more about the [`Manage Data` operation](../../../../docs/glossary/operations/manage-data.mdx).

<AttributeTable>

- OpSuccess
  - MANAGE_DATA_SUCCESS
  - Manage data operation has executed successfully
- op_not_supported_yet
  - MANAGE_DATA_NOT_SUPPORTED_YET
  - The network hasn't moved to this protocol change yet
- op_data_name_not_found
  - MANAGE_DATA_NAME_NOT_FOUND
  - Trying to remove a Data Entry that isn't there
- op_low_reserve
  - MANAGE_DATA_LOW_RESERVE
  - not enough funds to create a new Data Entry
- op_data_invalid_name
  - MANAGE_DATA_INVALID_NAME
  - Name not a valid string

</AttributeTable>

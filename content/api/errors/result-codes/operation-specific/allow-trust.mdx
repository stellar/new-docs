These are result codes that communicate success (200) or failure (400) specific to the `Allow Trust` operation.

Learn more about the [`Allow Trust` operation](../../../../docs/glossary/operations/allow-trust.mdx).

<AttributeTable>

- OpSuccess
  - ALLOW_TRUST_SUCCESS
  - Trust trust operation was successful
- OpMalformed
  - ALLOW_TRUST_MALFORMED
  - asset is not ASSET_TYPE_ALPHANUM
- op_no_trustline
  - ALLOW_TRUST_NO_TRUST_LINE
  - trustor does not have a trustline, source account does not require trust
- op_not_required
  - ALLOW_TRUST_TRUST_NOT_REQUIRED
  - source account does not require trust
- op_cant_revoke
  - ALLOW_TRUST_CANT_REVOKE
  - source account can't revoke trust
- op_self_not_allowed
  - ALLOW_TRUST_SELF_NOT_ALLOWED
  - trusting self is not allowed

</AttributeTable>

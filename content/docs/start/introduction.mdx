Stellar is a public distributed ledger for near-instant payments and universal currency conversion. It's very developer-friendly, and  allows you to connect products, applications, and existing financial infrastructure.  The developer docs explain how to roll up your sleeves and get building on Stellar, and will help you understand key concepts and practical applications of the network.

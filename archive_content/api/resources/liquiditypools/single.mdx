The single liquidity pool endpoint provides information on a liquidity pool.

<Endpoint>


|     |                                     |
| --- | ----------------------------------- |
| GET | /liquidity_pools/:liquidity_pool_id |

</Endpoint>


<AttributeTable>


- ARGUMENT
  - REQUIRED
  - DESCRIPTION
- liquidity_pool_id
  - required
  - A unique identifier for this liquidity pool.

</AttributeTable>


<CodeExample title="Example Request">


```curl
curl "https://horizon.stellar.org/liquidity_pools/67260c4c1807b262ff851b0a3fe141194936bb0215b2f77447f1df11998eabb9"
```

```js
var StellarSdk = require("stellar-sdk");
var server = new StellarSdk.Server("https://horizon.stellar.org");

server
  .liquidityPools()
  .liquidityPoolId(
    "67260c4c1807b262ff851b0a3fe141194936bb0215b2f77447f1df11998eabb9",
  )
  .call()
  .then(function (resp) {
    console.log(resp);
  })
  .catch(function (err) {
    console.error(err);
  });
```

</CodeExample>
